# OpenClaw x Haravan

> **Trợ lý vận hành cửa hàng Haravan** — Dùng AI Agent quản lý đơn hàng, tồn kho, khách hàng bằng tiếng Việt tự nhiên.

## Dành cho ai?

- Chủ shop Haravan muốn AI kiểm tra đơn hàng, tồn kho mỗi sáng
- Nhóm vận hành (Ops) cần dashboard tự động bằng ngôn ngữ tự nhiên
- Marketer muốn phân khúc khách hàng VIP, tìm khách cần reactivation

## Cài đặt nhanh

```bash
export HARAVAN_TOKEN="YOUR_TOKEN"
npx @haravan-master/openclaw-haravan-ops-mcp
```
