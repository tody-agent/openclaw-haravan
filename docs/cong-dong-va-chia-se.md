# Cộng đồng & chia sẻ

## Link nào để gửi bạn bè / khách?

- **Trang chủ:** [/](/) — thấy giá trị và câu hỏi mẫu ngay.
- **Bản đồ tài liệu:** [/bo-tai-lieu](/bo-tai-lieu) — chọn đúng mục tiêu.

Site được build từ mã nguồn mở; chi tiết kỹ thuật (hosting, build) dành cho maintainer — xem repo GitHub bên dưới nếu cần.

## Repo & vấn đề (issues)

- **Mã nguồn:** [github.com/tody-agent/openclaw-haravan](https://github.com/tody-agent/openclaw-haravan)
- Bug, ý tưởng playbook, cải thiện docs → **GitHub Issues** (mô tả repo + bước tái hiện).

## Góp ý & đóng góp

Ý tưởng, lỗi docs, câu hỏi mẫu hay → **[GitHub Issues](https://github.com/tody-agent/openclaw-haravan/issues)**. Người chỉnh sửa mã có hướng dẫn trong repo (không cần để mọi người dùng đọc).

**Cài plugin (người dùng):** [Plugin Haravan Ops](/plugin-openclaw) — lệnh cài: `openclaw plugins install @haravan-master/openclaw-haravan-ops-plugin`.

## Nguyên tắc khi demo công khai

- Dùng **shop demo** hoặc dữ liệu đã ẩn danh.
- **Không** stream token hay response chứa PII khách hàng thật.
- Nhắc audience: thuế / P&L trong tool là **ước tính** — [Disclaimer](/lean/safety-disclaimers).
